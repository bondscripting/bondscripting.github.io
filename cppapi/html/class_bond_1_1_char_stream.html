Detailed Description

An iterator over the contents of a string of text.

CharStream has methods to return the next available character, peek at the next character, push characters back into the stream and to query the position of the stream.

CharStream is used internally by Lexer; it is unlikely that someone integrating Bond into their application will have to make direct use of it.
