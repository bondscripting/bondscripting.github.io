Detailed Description

Represents a collection of related TokenTypes.

A TokenTypeSet is used to validate that type of a Token belongs to a collection of related TokenTypes. It is typically used by a Parser to validate that the Tokens contained in a TokenStream are syntactically correct.

See also
Parser, TokenStream
