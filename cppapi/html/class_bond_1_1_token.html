Detailed Description

A token describing a syntactic unit of the Bond scripting language (e.g. keywords, identifiers, literal constants and operators).

A Token is typically created by a Lexer as it scans Bond source code. It contains the text from which the token is formed as it appears in the Bond source, the position at which it appears in the Bond souce, a type and a value for tokens which are literal constants.
