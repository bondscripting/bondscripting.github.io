Home of the Bond Programming Language.

Bond is a lightweight embeddable scripting language implemented in C++ and its source code is distributed for free under the permissive MIT license. Bond is very similar to C in many respects including syntax, static typing and explicit dynamic memory management.

This website is still under construction, so please stay tuned for additional documentation. In the meantime, the source code for the Bond runtime library and command line toolset can be found on GitHub at the folllowing link: https://github.com/bondscripting/bond

Or download the source code from the command line with the following command:

git clone https://github.com/bondscripting/bond.git
